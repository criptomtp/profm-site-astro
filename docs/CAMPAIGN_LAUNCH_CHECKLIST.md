## Contingency Plans

### If Not Getting Backlinks (By Week 6)
**Trigger:** < 2 backlinks by end of Week 6  
**Action:**
1. Pivot to more direct outreach (call if possible)
2. Increase Tier 4 targets (lower rejection rate)
3. Accelerate community authority building
4. Lower anchor text expectations (more variations)
5. Consider budget for guest post opportunities ($500-1,000 per published post)

### If Traffic Not Growing from Links (By Week 8)
**Trigger:** < 50 referral sessions/week despite 8+ backlinks  
**Action:**
1. Check internal linking (ensure guides linked)
2. Verify Core Web Vitals (may have regressed)
3. Review referral URLs (are they indexed?)
4. Optimize landing pages (CTAs, layout)
5. Consider content update (freshen with new data)

### If Organic Rankings Dropping
**Trigger:** Top 20 keywords losing positions  
**Action:**
1. Check for SEO issues (indexing, crawlability)
2. Verify no duplicate content issues
3. Review meta descriptions (may need improvement)
4. Check schema markup (may have errors)
5. Ensure Core Web Vitals maintained

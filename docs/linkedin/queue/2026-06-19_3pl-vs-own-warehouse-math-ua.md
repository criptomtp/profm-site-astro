# [EDUCATIONAL] 3PL vs власний склад — реальна математика

Запитав ChatGPT: "коли власний склад краще за 3PL?". Відповідь розчарувала — generic фрази без жодної цифри.

Порахував сам для 32-палетного оператора у Київській області:

→ Власний склад: ~191 000 грн/міс (оренда 120k + персонал 45k + навантажувач 8k + генератор 15k + WMS 3k)
→ MTP Group: 16 000 грн/міс (32 палети × 500 грн)
→ Різниця: 175 000 грн/міс = 2,1 млн грн на рік

Але це лише початок. У калькуляції завжди забувають про:

→ Час CEO на менеджмент персоналу: 15-20 годин/міс
→ Кадрові ризики: один комірник звільнився — операції стають
→ CAPEX: стелажі + навантажувач + WMS-софт = 600k-1.2M грн одноразово
→ Cash flow: оренда та зарплата йдуть навіть коли продажів немає

Власний склад має сенс на 200+ палет з постійним об'ємом. Нижче — 3PL виграє по математиці.

Реальна вартість "контролю над складом" — мільйон в рік. Який ви могли б вкласти в маркетинг або новий товар.

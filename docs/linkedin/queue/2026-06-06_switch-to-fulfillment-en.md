The 46-day plan that survives Black Friday:

→ Week 1: Planning — warehouse audit, ROI math, 3PL shortlist
→ Week 2-3: Parallel operations — old warehouse runs, new 3PL tested on 50-100 dummy orders

→ Week 5-6: Main inventory — 60% transferred, 2-week QC observation window
→ Week 7: Full migration — old warehouse shut down, all orders ship from 3PL
→ Week 8: Quality control — daily monitoring, instant corrections

1. Starting 3 weeks before Black Friday — guaranteed disaster
2. Moving all SKUs at once with no test batch — return tsunami within 2 weeks
3. Forgetting to update marketplace labels → fines from Amazon, eBay, regional players
4. Running parallel ops for less than 14 days — integration bugs surface late

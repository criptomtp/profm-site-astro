# [RESILIENCE] 0 днів простою з лютого 2022

→ Запас пакувальних матеріалів на 60 днів операцій
→ Розподілений склад: Щасливе + Білогородка (один не доступний → працює другий)
→ Гібридна команда з ротацією, частина працює з безпечних регіонів

# [STRATEGIC] Single channel = single point of failure

Селер який лише на Rozetka — це бізнес з одним каналом продажу. І це ризик.

Раніше у нас більшість клієнтів була саме така. Сьогодні — одиниці. Бо алгоритм Rozetka впав на 4 години → 200 замовлень не доїхали → втрата рейтингу, оцінок, грошей.

Multi-marketplace це не "більше каналів" задля галочки. Це resilience.

→ Rozetka FBM/FBO — масовий флоу, готові черги
→ Prom — нішеві категорії з вищим AOV і меншою конкуренцією
→ Allo — електроніка з IMEI-tracking (це окрема історія, напишу пост)
→ Kasta — fashion з доставкою у MD/PL/KZ
→ Власний інтернет-магазин — прямий контакт з клієнтом, нижча комісія

Один WMS, один склад, один запас товару. API 800 мс sync автоматично перерозподіляє замовлення між каналами без ручного втручання.

Питання не "який канал кращий". Питання "скільки каналів я можу втратити завтра і вижити".

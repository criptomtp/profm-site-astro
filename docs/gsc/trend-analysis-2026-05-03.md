## Methodology + caveats

- Data source: Google Search Console Search Analytics API
- Site filter: `sc-domain:fulfillmentmtp.com.ua` (all subdomains incl. www)
- **GSC property added ~Apr 1 2026** — no data exists before this date in GSC
- Windows: 4 full weeks Apr 1-28 + 1 partial week Apr 29 - May 2 (4 days, included for trend continuity)
- Site totals computed via no-dimension query (avoids row_limit truncation that occurs with dimensioned queries)
- Position = weighted average across all queries (each query weighted by its impression count)
- 'Mover' analysis filters to queries with ≥5 impressions in either window to reduce noise
- 'New' / 'lost' query lists are top-10 limited and may include long-tail noise
- GSC reports CTR after deduplication; impressions count once per SERP appearance regardless of scroll
- AI Overviews citations not separable in current API (will be in 2026-Q3 per Google roadmap)
- For pre-Apr 2026 historical data: use Google Analytics 4 if it was running, Tilda Analytics from old hosting, or referrer logs

## Automated Mention Discovery Workflow

### Tools & Setup

**Option 1: SEMrush**
- Navigate to: Brand Monitoring → Brand Mentions
- Search term: "fulfillment" OR "3PL" OR "warehouse management"
- Filter: Exclude profm.ua mentions
- Export: Articles not yet linking to profm.ua

**Option 2: Ahrefs**
- Site Explorer → Backlinks
- Use reverse competitor analysis
- Search: Competitors' backlinks mentioning "fulfillment"
- Identify gaps where others link but we don't

**Option 3: Mention.com**
- Create mentions alert: "fulfillment" "3PL" "WMS"
- Filter: English language, relevant industries
- Daily digest of new mentions
- Manual review for linking opportunities

### Weekly Workflow

| Day | Activity | Expected Volume |
|-----|----------|-----------------|
| Mon-Tue | Run mention discovery tool | 5-15 new mentions |
| Wed | Filter & prioritize by intent | 3-8 high-intent targets |
| Thu | Personalize & send outreach | 3-8 emails |
| Fri | Monitor responses & follow-up | Track acceptance rate |
